以下是在研究 CSS 各種光怪陸離的問題和效果的路上，搜集到一些用來查找資料，或是觀摩大神範例的資源：


## 網站

[CSS-Tricks - Tips, Tricks, and Techniques on using Cascading Style Sheets.](https://css-tricks.com/)

:::note
CSS-Tricks 我會用來看一些新語法的介紹和範例寫法，基本定義可以查找 [W3C](https://www.w3schools.com/cssref/index.php) 或 [MDN](https://developer.mozilla.org/zh-TW/)，但如果是想要看一些基礎到進階的實作應用，或是一些酷炫效果的範例，我就會來這邊挖挖寶。

當然網站裡不僅僅只有 CSS 主題，還有許多跟前端相關的其他主題，文章更新的頻率和質量都很好。
:::


- [Smashing Magazine — For Web Designers And Developers](https://www.smashingmagazine.com/)
- [Codrops](https://tympanus.net/codrops/)

:::note
Smashing Magazine 和 Codrops 也是我會拿來觀摩一些特殊效果、或是查找進階應用和新知識的資源。

有些實作不見得能完全解決自己遇到的問題，畢竟實際開發產品的過程中，還是會遇到向下相容性或是客戶需求等限制，最新技術最酷炫的效果往往到最後會被閹割或是(取)捨，在看這些效果範例時，也開始會思考如何兼顧效果和彈性/可讀性，這兩個網站的文章更新的頻率和質量也都很不錯，適合挖寶。
:::
